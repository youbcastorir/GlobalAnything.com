GlobalAnything PWA Icons
========================

Place your app icons here with these exact filenames:

  icon-72.png   (72x72)
  icon-96.png   (96x96)
  icon-128.png  (128x128)
  icon-192.png  (192x192)  ← Required for Android install
  icon-512.png  (512x512)  ← Required for splash screen

How to generate icons:
1. Create a 512x512 PNG of your logo
2. Visit https://maskable.app to make it maskable
3. Visit https://realfavicongenerator.net to generate all sizes
4. Place the output files in this folder

The icon should use the GlobalAnything teal color (#14b8a6) as background
with a white keyboard (⌨) or globe symbol centered.
